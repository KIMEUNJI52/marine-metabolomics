## 0. 프로젝트 개요

대한민국 **연안 해양환경**을 배경으로, 해양 인근 **자생 나무·식물의 성장/변화를 대사체분석(metabolomics)**
으로 규명하는 연구 프로젝트. 그 기반 작업으로 **연안 표층수온(실측) 데이터 파이프라인과 시각화**를 구축함.

## 3. 데이터 집계 스크립트 (`src/`)

| 스크립트 | 역할 |
|----------|------|
| `aggregate_sst.py` | 연·월·해역별 월평균 집계 → `sst_monthly_by_sea.csv`, `sst_dashboard.json` |
| `aggregate_sst_stations.py` | 관측소별 (전체+연도별)×(연평균+월별) 집계 → `sst_stations.json` |
| `build_map_stations.py` | 관측수 충분한 46개 대표 관측소 선별 + 좌표 부여 → `sst_map_stations.json` |
| `build_combined_html.py` | 여러 대시보드를 단일 공유용 HTML로 통합 |
| `data_processing.py` | 대사체 전처리 유틸(결측·log·Pareto 스케일) |

집계 산출물: `data/processed/` (JSON/CSV)
